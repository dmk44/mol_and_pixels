# mol_and_pixels

Сопроводительный репозиторий к телеграм-каналу ["Молекулы и пиксели"](https://t.me/mol_and_pixels).

Здесь будут публиковаться:
- скрипты анализа;
- данные для примеров;
- иллюстрации и графики;
- расширенные версии разборов.

Репозиторий постепенно наполняется по мере выхода материалов.
